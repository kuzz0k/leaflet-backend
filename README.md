# Leaflet Tile Server

Бэкенд-сервер на Go для раздачи тайлов карт в формате `.mbtiles`.
Проект разворачивается через Docker на отдельной машине и хранится в gitlab репе, настроен CI/CD для создания образов и выгрузку в registry. 

### Стек
 - Go, PostgreSQL, Swagger, Docker, Docker-compose, Gitlab CI/CD, s3

### Прочие улучшения
- Переход на S3-совместимое хранилище
- Структурированное логирование: Внедрить `slog` или `zerolog` для записи логов в формате JSON.
- Тесты и Линтинг: Добавить этапы `test` и `lint` в CI/CD пайплайн.
- Автоматические миграции: Использовать `golang-migrate/migrate` для автоматического применения миграций базы данных при старте приложения.
